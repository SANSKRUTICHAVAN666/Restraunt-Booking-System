# DineBridge – Restaurant Discovery & Table Booking System<br>
## Overview
DineBridge is a web-based platform that allows users to discover restaurants, view menus, and book tables online. The system also provides restaurant owners with tools to manage menus, reservations, and customer bookings efficiently.

## Features
- Restaurant discovery and browsing
- Real-time table reservation system
- Digital menu viewing
- User authentication and login system
- Reservation tracking and management
- Dashboard for restaurant owners to manage bookings and menus


## Technologies Used
HTML5<br>
CSS3<br>
JavaScript<br>
Firebase Authentication<br>
Firebase Firestore Database<br>
Firebase Hosting<br>


## System Modules

### 1.User Module
- Register and login to the platform<br>
- Browse restaurants and view menus<br>
- Book tables and manage reservations<br>

### 2.Restaurant Owner Module 
- Add and manage restaurant information<br>
- Update menu items<br>
- Manage table bookings and reservations<br>

### 3.How to Run the Project
- Download or clone the repository.<br>
- Open the project folder.<br>
- Open the index.html file in a web browser or run using Live Server in VS Code.<br>
