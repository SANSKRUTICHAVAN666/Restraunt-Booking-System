#DineBridge – Restaurant Discovery & Table Booking System
This  system allows users to discover restaurants, check real time table availability, book tables and browse menus as well as check the ingredients of the particular dish in a menu. The system also enables owners to manage menus, reservations and orders through a dashboard improving efficiency, customer waiting time and transparency of the menu. 


#Features
Restaurant discovery and browsing
Real-time table reservation system
Digital menu viewing
User authentication and login system
Reservation tracking and management
Dashboard for restaurant owners to manage bookings and menus


#Technologies Used
HTML5
CSS3
JavaScript
Firebase Authentication
Firebase Firestore Database
Firebase Hosting


#System Modules

1.User Module
Register and login to the platform
Browse restaurants and view menus
Book tables and manage reservations

2.Restaurant Owner Module
Add and manage restaurant information
Update menu items
Manage table bookings and reservations

3.How to Run the Project
Download or clone the repository.
Open the project folder.
Open the index.html file in a web browser or run using Live Server in VS Code.
